- 赫伯特·西蒙曾因其在经济学理论方面的建树而荣获了1978年的诺贝尔奖。他仔细考查了他本人所做的著名的科学探索的过程(Simon, 1989)。在所有的个案中，他发现，科学探索的方法可以用我们在认知心理学中所研究的基本的认知过程来阐明。他写道：“此外，被认为是从事科学发现这类工作所必需具备的顿悟，则被证明是“识别”这一我们所熟知的过程的同义语。通常用于讨论创造性工作的其他术语，例如“判断力”、“创造力”，甚至“天才”等，看来统统都是没有必要的，或者是，如同顿悟那样，可以用熟知的寻常概念来定义(p. 376)，”西蒙得出结论，当我们仔细考察人类智慧时，我们发现，它涉及的是基本的认知过程，
- （基本感知）
	- 视觉
	  collapsed:: true
		- 人类的感觉信号中有约80%是通过视觉获得的，然而这一能力也存在着限制，
		- 视觉拥挤效应：指在存在于外周视野内的目标周围增加几个侧翼干扰物时，目标会更难辨认的现象。
	- 听觉
	  collapsed:: true
		- 一般认为，人类能听到的声音频率范围是20到20，000赫兹，但这只是理想数值，一般人比较敏感的频率区间为1000到4000赫兹，
		- 现在认为，听觉皮层整合了关于动物当前感知和行为的信息，以便产生与动物当前状况相关的听觉信号，而并非机械的反应声音，
		- 鸡尾酒会效应：人类可以在嘈杂的声音环境中听清希望听清的言语；相比之下，在有多个说话人的条件下，机器识别某个说话人言语的正确率非常低。
		- 部分研究者认为，这可能是由于每个声源的声音特性具有一定差异，听者可以利用这些与目标语句相关的各种知觉线索来觉察、捕捉和追随目标语句流；此外，对发出声音的目标的空间位置的了解，也可以帮助听者选择性的注意集中发出声音的目标，
		- （部分研究显示，音乐训练对大脑发育有益，但只有系统的长期的音乐训练，而且也要在年岁很小的时候就开始的训练才有一定效果），
	- 触觉
	  collapsed:: true
		- 体感系统：包括三大部分，外部感受系统（分析皮肤收到的刺激信号）、内部感受系统（分析肌肉，平衡器官等身体内的刺激信号）、感觉传递系统，
		- 研究没有找到处理疼痛感的具体大脑皮层——虽然岛叶、丘脑和前扣带回皮层都会在感受到疼痛时被激活，但它们都不是感受疼痛的必要条件；然而，前扣带回皮层与疼痛的相关较为显著，
		- 不同于其他感觉，疼痛可以被认知和情感因素抑制，生理学研究显示，镇痛药物和心理因素可能都是通过从中脑的内啡肽回路来阻断疼痛感的；前扣带回皮层和前额叶皮层也在这一过程中发挥了一定的作用，
	- 嗅觉（味觉）
	  collapsed:: true
		- 嗅觉和味觉被称为化学感官，因为它们的功能是“监测”环境中的化学成分，
			- 嗅觉是嗅觉系统对通过鼻道中的感受器吸入的空气中的化学物质的反应，
			- 味觉是味觉系统对口腔中溶解的化学物质的反应，
			- 当我们吸气时，嗅质可以随着吸入的空气进入鼻腔（鼻前通路），在我们进食时，食物所释放的嗅质也可以随着呼出的空气从口腔进入鼻腔（鼻后通路）；这两种方式都能有效地诱发嗅觉反应。我们进食所感受到的“味道”主要来自于鼻后通路的嗅觉体验，
		- 功能
			- 进化历程中最早形成的感官，
			- 一般认为嗅觉的主要功能为寻找食物，并避免有毒物质，
			- 嗅觉的其他功能包括生物间的信息交流，即生物通过信息素互相辨认；然而并未发现与人类有关的信息素，
			- 相比之下，嗅觉与情绪的关系紧密，嗅觉是唯一一个不需要经由丘脑中转就可直接投射至大脑皮质的感官；同时，嗅觉刺激与情绪信息的加工都直接涉及边缘系统。研究显示，不同的味道可以调节人类的情绪状态，反之，情绪对嗅觉感受也有一定影响。
	- 感知的能动性
	  collapsed:: true
		- 应该指出，我们并非被动的对外界事物做出感知，我们最终形成的感觉是建立在我们的已知知识之上的能动反映；从生理学角度上，即我们最终接受的信息是大脑各个部位对感官接受到的信息进行加工之后的产物，
		- 此外，我们也有能力专注于我们希望得到的信息；脑成像研究显示，前额叶皮层和后顶叶皮层在引导注意力方面起着主要作用，
- 认知
  collapsed:: true
	- 信息的复杂性
		- 信息过载
			- 信息的数量与信息之间的联系
			- 在一定的时间内，我们能“理解”和“应用”的信息是有限的，过多的信息反而会让消费者无法做出最佳决定。此外，信息间的联系很广泛，我们很难明确判断信息是彻底无用的，
			- 编码新颖、陌生的事件需要占用很多认知资源，但新的事件过多会导致我们没有充足的时间“理解”这些新信息，
		- 接受新信息的必要
			- 吸收新概念、学习新事物可以延缓阿尔茨海默症；保持积极的心态，做一些从未做过的事情，这可以让血液输送到大脑之前没有得到血液的部分，延缓老化效应，
			- 并不是我们需要接受更少的信息， 而是我们需要有一个系统来组织它们。
		- 应对
			- 主动选择：从信息的来源入手，筛选“更可能”有用的信息发布者或发布平台，
			- 放弃信息：人们往往会担心最关键的信息下一次就会出现，但我们并不能充分利用过多的信息，因此应当学会放弃信息，
				- 研究显示，首先完整分析前1/3的选择，然后在遇到下一个更好的选择的时候停止，这样的胜算最高（Gilbert/Mosteller (1966) Recognizing the Maximum of a Sequence），
			- 训练注意力
				- 把注意力集中在呼吸上，借助腹式呼吸，你可以激活副交感神经系统，降低你的心率，使你平静下来，还会激活平抑杏仁核的神经系统。通过关注呼吸，你也可以消除由担忧所造成的紧张情绪。
				- 结合瑜伽的冥想可以实现身心平静，这些伸展运动能促使健康的血液流向大脑，让人注意力更集中、精神更放松。可以花一点儿时间，甚至在工作间隙，通过深呼吸、伸展四肢和想象肌肉被拉伸的感觉来降低心率。一般练习几分钟就可以取得成效，
				- 应该接纳冥想时产生的想法，而不是强迫自己什么都不想，试图摒弃所有的想法的行为难以实现，且可能带来不必要的压力，
	- 信息的不准确性
		- 来源
			- 认识的不全面
				- 预先选择效应：部分实验在设计时就有一定的特殊要求，而且人们对于接收到的信息也有一定的个人选择，
				- 个人的偏见：作者自己的政治与社会观点，提供信息来源的公民的错误判断和政治倾向，
			- 认识的发展
		- 应对
			- 了解信息的提供者（政府、专家、支持者、业余人士）和信息的赞助者，以及这些人是否有明确的政治立场，
	- 信息的组织
		- 拥有多个账户可以帮助我们组织、划分事件，免受干扰，
		- 整理物品可以让人们的大脑得到很好的休息，因为大脑可以从生活空间中的大量事物中找到新的联系，并通过神游模式重新定义组织这些物品之间的关系，以及物品与我们之间的关系。此外，确定可以找到需要的物品可以我们节省时间，用来完成更加有意义的任务。从心理学的角度来说，这也减少了我们不清楚能否找到所需物品的压力，从而产生一种舒适感。
		- 无法从逻辑角度分类不代表我们的逻辑存在错误，或是我们没能认识这些事物；这只是因为事物之间的联系的普遍性与多样性，
- 记忆
  collapsed:: true
	- 记忆的类型
		- 短时记忆（工作记忆）
			- 背外侧前额叶皮层负责管理工作记忆。工作记忆与大脑中当时正在处理的事情相关联，只能短时间保留。
			- 工作记忆是长时记忆的基础，
		- 长时记忆
			- 长时记忆的存储似乎是一种动态的过程，而且遍布大脑的不同区域。
			- 外显性记忆
				- 情绪记忆
					- 包含了强烈的情绪的情景记忆被称为情绪记忆，
					- 消极信息更有吸引力、更容易记忆。这可能是因为积极信息通常验证了我们所知道的，而消极信息通常涉及我们不知道的。
					- 情绪化的事件会激起我们的生理反应，推动记忆的巩固，情绪记忆的神经网络经常与恐惧体验联系在一起，即杏仁核会在情绪学习中起到重要作用。
				- 海马体主要负责为外显性记忆进行编码，它从前期的学习和信息中产生想法，
			- 内隐性记忆
				- 程序性记忆
					- 关于动作的习惯性方式，即如何重复特定的流程，包括阅读、漱口、打字、骑自行车等技巧和习惯。程序性记忆一般不涉及丰富的具体内容，而是促使我们下意识地完成不同的动作和流程。
					- 一般认为，小脑和纹状体在关于躯体运动的程序性记忆中起作用；然而，越来越多的证据表明，二者在某些只涉及思维的程序性记忆中也有一定的作用。
					- 在人的后半生，海马体会逐渐萎缩。人们会逐渐难以记住当下发生的事情，但仍存在程序性记忆。
	- 记忆的机制
		- （内侧）颞叶皮层：包括海马体及其周围的大脑皮层，其中海马体的主要功能是在信息间建立联系，而其它区域则的主要功能则是进一步加工信息，并促进短时记忆向长时记忆转化，
		- 前额叶皮层：主要与工作记忆相关，也参与了对内隐记忆的启动，
		- 间脑系统：一些案例显示，间脑系统受损的病人也在回忆发生的事件方面存在障碍，但间脑在记忆中的作用并没有上述脑区明显，
	- 记忆的巩固
		- 长时程增强效应（LTP）
		- 主观意识的参与
			- 记忆就是分散在大脑各处信息片段的集合，而海马体可以从皮层接收信息片段，并将这些片段组合为整体的记忆；
			- 然而，人们需要有意识的利用前额叶皮层来促使海马体参与这一行为，
		- 睡眠
			- 新获得的记忆一般是不稳定的，需要一个增强和巩固的过程；大脑会分析和分类新的信息，与已有的认知结构、已存储的信息整合，
			- 这些不同的过程在睡眠的不同时段完成，而且需要几个连续的夜晚，
			- 睡眠可以分为快速眼部运动（REM）和非快速眼部运动（NREM）两大周期，减少轻度睡眠、改变睡眠习惯、睡眠时长不足、饮酒和药物（包括安眠药）都可能打乱睡眠周期，从而干扰我们的记忆，
			- （在经历创伤后应保持清醒，避免睡眠，以防止睡眠对创伤记忆的巩固作用），
		- 检索记忆
			- 我们的生活不是完全一样的时刻，而是大脑在下意识地将不同时间像电影一样剪辑、分块，即自动给时间设置一个开始和结尾；
			- 分块可以在需要时进一步细化；例如下午可以分为午餐、睡觉、工作；午餐可以细分为准备、吃饭、整理；准备可细分为洗菜、切菜、做菜等，
			- 大脑分类出的时间块不一定是清晰的，所以大脑可能经常自动填补缺失的信息，
	- 记忆的破坏
		- 少量的皮质醇促进记忆的形成，而大量的皮质醇会抑制记忆，皮质醇超负荷时，会毁坏神经元之间的连接，破坏记忆；
		- 研究显示，抑郁症患者的海马体会缩小，且萎缩的程度与患抑郁症的时间相关，也有研究观察到抑郁状态会导致大脑的其他区域的突触停止生长，从而阻断神经元的交流。这可能导致患者更难以理解和记忆新的事物，而停留于过去的（消极）记忆也会使得患者的大脑更容易变得消极，
	- 记忆的不准确
		- 大脑会主动将类似的事物归为一类，因此“常规”的记忆可能难以记清，
		- 一个想法可能会激活其他记忆，使得记忆出现错误；此外，大脑可能会检索出事件的多个历史版本，这些记忆混在一起，使得大脑很难找到相关的那一个，
	- 外部记忆机制
		- “吉布森可供性”：将大脑组织负担转移到外部世界，如物体的特殊设计，或将物品尽量放在最可能需要他们的位置；即通过外部实体记录信息，减少对记忆的依赖，
			- 例如，预报要下雨时将雨伞提前放在门口；大脑习惯了没有改变的东西，因此不会注意一直放在门边的雨伞，即使有预报可能要下雨；将关键的东西设立备份，并放置在不同位置；将重要的东西分类存放；
			- 外部控制行为信息：襟翼控制的设计看起来像一个微型皮瓣；起落架控制设计成了圆轮，类似于起落架的样子。飞行员不用再依靠他们的记忆记住这两个操作控件，操作控件本身就能提醒飞行员它们各自的用途，
		- 信息外化并不总是需要将信息写下来，或者使用其他外部媒介进行解码，有时外界本身就存在着规则，你只需要读懂这些标记。例如高速公路的数字编号，
- “学习”
  collapsed:: true
	- 定义
		- 理论定义：分析材料并找到规律，检查规律是否符合材料，并依据规律去预测新的事件，
		- 实践定义：可以在下一次面对类似问题时有更好的表现，
	- 归纳
		- 人类在解决问题时，往往是先过分地概括化，发现错误后再把概括程度降低，或从其他方面找到合适的概括水平。
		- 我们得到正确结论的最终过程是演绎推理，但实际上其中的每一步都是一次归纳搜索，
		- 在进行模式归纳的过程中，第一要产生一个规则，提出模式的假设；第二，应用这个规则把具体的实例加以归类；第三，根据对实例进行归类的结果所得的反馈信息，提出规律，知道最初提出的假设是否正确。
		- 其它
			- 科学家先收集大量的材料，然后分析这些材料，找出规律性的东西，再解释这个规律，这种过程被称为“材料驱动归纳”。科学家先有理论，然后根据理论进行预测，看事实是否符合理论。最后由同行、专家评论这种理论是否正确。这种过程被称为“理论驱动归纳”。理论驱动归纳是在已知理论的指导下探索规律，理论能限制科学研究的范围，减小搜索的空间，因此往往更有效。
	- 练习
		- 为达到重塑大脑的目的，你必须使新行为保持足够长的时间，直到新行为变成你下意识的动作。假以时日，不断实践就会让你毫不费力。一旦达到这个水平，你的大脑就不会很辛苦。
		- 正电子发射断层成像的扫描图片显示，一个人的某种技能越熟练，大脑中与此技能有关的区域付出的劳动就越少，
		- 直觉实际上是一种再认，一个人只有对非常熟悉的东西才会有直觉。通常所说的科学上的偶然发现，对专家来说却不是偶然的，是他利用了已有的知识认识到了当前的情景。
	- 抽象与具体
		- 通过例题学习，
		- 把所学的东西做某些改变才能用于解决新的问题，
- 问题解决
  collapsed:: true
	- 问题表征
		- 模式识别：人要建立事物的模式，就必须认识元素之间的关系。元素之间有各种关系，例如等同关系、连续关系等。根据元素之间的关系，就可以构成模式。解决问题和认识系列模式之间具有共同性，这在科学发现的研究中可以得到应用。
		- 用简明的图解提供视觉形象，对于解决困难问题是有很大帮助。视觉表象可以帮助我们看出根本不可能的问题，
		- 选择性地关注与完成任务最相关的特征，忽视那些会争夺注意力的环境特征与刺激；这需要形成主动的习惯，也需要相关经验的积累，
		- 问题分割
			- 日常生活中的一些现实问题和专业问题，并不那么清楚明确，问题的要求可能在不断的变化。
			- 然而，结构不合理的问题实际上是由一系列结构合理的小问题组成的。正因为如此，结构不合理的问题才能够被解决。在解决小问题的每一个阶段里，也提出了下一个待解决的新问题，同时对整个问题的解决也提供了信息。
			- 可以通过制定计划，把整个大问题中有助于解决问题的成分抽象出来。
			- 解决问题的次序也很重要，尤其是对那些影响较大的小问题来说，其解决的先后就显得更重要。如果设计得好，后期小问题的解决就不会与先前的小问题发生矛盾，更不至于改变整个设计方案。
			- 此外，任何方案都有相互矛盾的要求，总要进行取舍。可以只找到一个比较满意的解决办法，而无需保证这个解决方案是最好的，
	- 步骤记录
		- 短时记忆的容量有限，光凭短时记忆，被试只能记住上一个样例对不对，而无法利用以前的信息。如果给被试一张纸，让他记下样例的对错，这就会帮助他使用更有效的策略。
	- 手段-目的分析
		- 解题者通过当前情况与目标的不断比较而找到一个更好的算子。
		- 不应过分追求完美，也应考虑到整体任务的大局，即不能在细节的任务上花费过多的时间，
		- 为了确保最后的完成， 每个任务都应该有停下来审视的环节，
	- “创新”
		- 定义：将本没有联系的新事物联系在一起，或通过对已有信息的修改创造出新的信息，
		- 机制：左脑擅长精确的记忆已有的的知识，并对其进行排列组合；右脑的信息精度不足，但且更擅长连接不同事物，
		- 主动的创作会直接调动大脑的神游模式，刺激不同想法之间的自由组合与结合，在概念与神经元节点之间建立起某种连接。
		- 适当的随机浏览可以开拓我们的眼界，让我们发现全新的联系，然而信息如果发散的过于宽泛，则会失去内在联系，
	- 问题探索
		- 费米问题（估算）
			- 现实世界的很多问题都没有精确的解答；对于部分问题，我们或许能寻找到一些精确的表象，但这些精确值并不能帮助指导实践，合适的抽象与近似可能更有帮助，
			- 问题分割是解决这些问题的一种方式，将问题分成几个部分，分别列出我们需要知道的，然后再列出我们的估算。
			- 设置边界条件：综合分析已有的知识，设置一个大概的“边界条件”，例如尝试找出数字问题的答案的位数，或者一个很宽泛的上下界限，
